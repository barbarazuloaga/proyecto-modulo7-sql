# Proyecto Base de Datos SQL

Este proyecto consiste en el diseño y manipulación de una base de datos relacional utilizando SQL.  
El objetivo es aplicar conceptos fundamentales de creación de tablas, inserción de datos y consultas.

## Estructura del proyecto

El repositorio contiene los siguientes archivos:

- script_sql_comercioya.sql → Script SQL con la creación de tablas, inserción de datos y consultas.
- documento_breve_modulo7.pdf → Documento explicativo del desarrollo del proyecto.
- documento_breve_modulo7.docx → Versión editable del documento.

## Base de datos

La base de datos representa un sistema simple de gestión comercial con las siguientes tablas:

- **clientes**
- **productos**
- **ventas**

Estas tablas están relacionadas mediante claves primarias y claves foráneas.

## Funcionalidades implementadas

- Creación de tablas con SQL
- Inserción de registros
- Consultas de datos
- Relaciones entre tablas
- Modificación de estructura de tabla
- Eliminación de registros

## Objetivo del proyecto

Aplicar conocimientos de SQL para el diseño y manipulación de una base de datos relacional en un caso práctico.
